# gulp-template

This repository with base settings for web development website.

```bash
git clone https://github.com/WindForce225/gulp-template.git
```

## Starting Development

For MACos

```bash
sudo npm install --global gulp-cli
```
For Windows

```bash
npm install --global gulp-cli
```
For all

```bash
npm install
```

```bash
gulp
```

## Create build

```bash
gulp build
```
